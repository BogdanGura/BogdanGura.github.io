Bogdan's Academy

Bogdan’s Academy is a learning platform where you can learn coding with the help of articles and quizzes. The system would also allow me and my classmate to develop Bogdan’s Academy for the BPA 2024 Virtual Event Competition. We have received 7th place in the competition on the national level for your work. You can learn more about Bogdan’s Academy from our slideshow or see it for yourself by clicking the "Visit" button below.

What have I learned

Bogdan’s Academy was the most difficult project I have ever participated in. Its difficulty lay in the fact that it had to have a server side to work and accomplish the required functions. We used SQL and PHP for the backend and HTML, CSS, and JS for the front end. It was quite difficult to learn everything regarding PHP and SQL from scratch since we went the self-taught route. But in the end, everything worked out, and I’m glad I participated in the project because it gave me a lot of experience on the server side of programming.

Tetris Enhanced

To experience all the content you need to have 1920 * 1080 resolution

Tetris is a game where the player controls blocks, or tetrominoes," falling from the top of a screen. The player's objective is to align these tetrominoes in a continuous horizontal line that, when complete, will disappear and give the player points. When tetrominoes stack on top of each other and reach the ceiling, then the game ends. In Tetris Enhanced, players can play in solo mode or against an AI. There are also plenty of achievements in the game to be earned and discovered. The player can control tetrominoes with either arrow keys or WASD keys. Also, when a game starts, there are buttons that represent arrow keys and are there for Tablet and Ipad users.

What have I learned

Tetris Enhanced was my second serious project. While developing it, I learned a lot about array functions, filtering arrays, DOM manipulation with JS, and expanding the achievement system.

Full Stack Clicker

Not compatible with mobile devices
To experience all the content you need to have 1920 * 1080 resolution

Full Stack Developer Definition: is a developer or engineer who can build both the front end and the back end of a website. The front end (the parts of a website a user sees and interacts with) and the back end (the behind-the-scenes data storage and processing) require different skill sets.

Full-Stack Clicker is a clicker style game that simulates a process of learning to become a Full-Stack Developer and earn money. At the start of the game you can pick the difficulty which will determine how fast you progress through the game. Player's goal is to reach the money objective that is stated in the difficulty. But to earn money you need to earn XP so you can learn skills to build cool projects that can increase your money.

What have I learned

I have gained greater experience working with interval and other time functions, creating level progression systems in my clicker game with items appearing when a certain condition is reached.

Snake Game Enhanced

Not compatible with mobile devices
This game is best viewed in full screen mode (PRESS F11)

Snake Game is a game where you control a snake with your arrows or WASD keys to avoid collision with the board's walls and eat apples to acquire points that randomly spawn on the board.

In the enhanced version, there are many achievements that a player can unlock while playing the game. Also, another item is added: mine. When a snake collides with a mine, the game ends the same as when the snake collides with a board border. The player can control the snake with either arrow keys or WASD keys.

What have I learned

Snake Game Enhanced was the first serious project that I have undertaken in my web development journey. Snake Game was the foundation of my other projects, like Tetris Enhanced, for example, since the achievement system was first created for Snake Game Enhanced to make my project stand out from other snake games. In conclusion, Snake Game was the project where it all began for me as a software developer, and I even got my classmates to go to my web portfolio and play it 🙂.
